- WordPress maintenance across multiple production sites

- WordPress hardening and vulnerability review
- Malware cleanup and suspicious user investigation

- Safe file edits through FTP, cPanel File Manager, Git, and staging environments
- Log review, HTTP headers, curl testing, and browser DevTools analysis
